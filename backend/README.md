# PlayCoreTV - Backend

Backend API for PlayCoreTV built with Node.js, TypeScript, and Express.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env` and configure:
```bash
cp .env.example .env
```

3. Configure Supabase credentials in `.env`

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Test

```bash
npm test
npm run test:watch
npm run test:ci
```

## Lint

```bash
npm run lint
npm run lint:fix
```
